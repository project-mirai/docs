      # 确定 SDK

      # JVM 平台 Mirai 开发

      Mirai 文档没有废话，请务必仔细阅读每一个字。仔细阅读文档花费的 1 分钟可能相当于不阅读文档而自己摸索解决办法的 1 小时。

      1. JVM 环境和开发准备工作（2 分钟）
      2. 配置项目依赖
        • 要把 mirai-core 嵌入一个应用使用，请阅读 配置项目依赖。
        • 要为 mirai-console (opens new window) 框架开发插件，请阅读 mirai-console 的配置插件项目 (opens new window)。
      3. 阅读 API 文档（见下文）

      如果你不知道 mirai-console (opens new window) 是什么，请阅读 Mirai 生态概览。

      如果你希望先确认 mirai 能够正常运行才能安心阅读文档，可克隆 mirai-hello-world (opens new window) 并运行其中 Kotlin 或 Java 入口点 main。
